배치 파일(git_reset.bat)을 실행할 때 시스템 환경 설정이나 권한 때문에 CMD 창이 실행되지 않아 오류가 발생한 것 같습니다.

배치 파일 대신, 사용 중이신 PowerShell 터미널 창에 아래 명령어들을 한 줄씩 복사해서 직접 실행해 주세요.

PowerShell 전용 명령어
1단계: 백그라운드 Git 프로세스 강제 종료
powershell
Stop-Process -Name "git" -Force -ErrorAction SilentlyContinue
(위 명령어를 입력해도 화면에 아무 메시지가 나오지 않는 것이 정상입니다. 프로세스가 정상 종료된 것입니다.)

2단계: Git 락 파일 제거
powershell
Remove-Item -Path ".git\index.lock" -Force -ErrorAction SilentlyContinue
(락 파일이 없는 경우에도 오류 없이 넘어갑니다.)

3단계: Git 상태 확인
powershell
git status
(위 명령어를 입력한 뒤 정상적으로 파일 변경 사항 목록이 출력되는지 확인해 주세요.)
