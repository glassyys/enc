p190872_local_chk_v05_gm.py 이소스 이용 아래 내용 종합하여 소스생성요청

- 1차요청

아래와 같은 작업하여 소스를 수정요청합니다.
1. 실행구조
프로그램 [검색기준테이블] [검색디렉토리] [mid 검색디렉토리 MID하위디렉토리] --db --conf D:\chksrc\mysql.conf

2. [mid 검색디렉토리 MID하위디렉토리] 조건
[mid 검색디렉토리 MID하위디렉토리] aaa,bbb,ccc 이런식으로나 여러개를 추가하면 
[검색디렉토리] 하위의 aaa,bbb,ccc 디렉토리하위의 모든 소스에 대해 검색
[mid 검색디렉토리 MID하위디렉토리] 지정하지 않으면 [검색디렉토리]하위 전체 검색

3. 소스검색방법
[검색기준테이블]의 column_name의 중복제거된 칼럼값 기준으로
[검색디렉토리] 지정한 mid 하위 디렉토리에서 소스 검색해서
파라미터로 입력한 테이블의 column_name과 단어가 일치하는 행 추출

4. 결과내용 
1) 해당소스전체기준 라인수
2) code 로 시작되는 결과
3) 행의 내용
5. 화면출력양식
화면에 아래와 같이 출력하면서 실행하되
소스검색 시 검색디렉토리하위 MID값 출력후 진행
mid가 없으면 검색디렉토리 맨 마지막 디렉토리면 기재
--------------------------------------------------------------------------------
-- 검색MID : MID
--------------------------------------------------------------------------------
[매칭] code로 시작되는 명령  스페이스 (검색기준컬럼명)
[내용] 
--------------------------------------------------------------------------------

7. 생성파일 기존생성파일 레이아웃유지하면서 추가된 항목 추가
1) query_text 추출 결과
2) 칼럼기준 추출행 결과자료

8. 7번에서 생성한 파일 테이블에 등록
테이블명은 프로그램명을 제외한 검색기준테이블을 시작으로 내용에 맞게 생성합니다.

9. 주석내용
1) 소스내용정리한 내용
2) 실행방식내용은 한줄로 작성후 주석처리
3) 수정이력표시


- 2차수정요청
검색기준테이블에서 column_name 추출조건은
옵션추가해서 
--where old 이면 "where tobe_enc_key <> '' and asis_enc_yn = 'Y'"
--where new 이면 "where tobe_enc_key <> '' and asis_enc_yn = 'N'"
옵션이 없으면 "where tobe_enc_key <> ''" 조건으로 조회하도록 변경요청합니다.

- 3차수정 요청
요청사항에 비해 불필요한 파일이 나오는데 아래와 같이 수정요청합니다.
1. 파일1과 파일2 파일3 모두 파일생성 및 테이블 생성생략
2. 파일명과 테이블명 앞에 p190872_를 붙여서 생성요청
3. 전혀값이 없는 불필요한 항목은 제외하고 생성하도록 수정
4. .txt파일은 mid별로 생성
5. 불필요한 로직은 생략하고 간단하게 해도되는데 요청기능은 없어지면 안됩니다.

--3차수정요청
그럼 실행결과에 출력하도록만 수정해주면 되고
1. 검색조건테이블의 레이아웃정정
db_name,tbl_name,column_name,type_name,integer_idx,mig_dec,tobe_enc_key,tobe_enc_rsn,asis_enc_yn 이니 이내용으로 수정
2. 칼럼기준 검색시 파라미터 옵션하나 추가
 --chk default 이면 "default.encrypt" 와 "default.decrypt" 이포함된 라인만
 --chk encdec_no  이면 "default.encrypt" 와 "default.decrypt" 이포함된 라인은 제외
옵션이 없으면 기존대로 모두 추출

--4차수정요청
1. 실행출력결과내용에 화면출력내용 파일 생성한 파일명(p190872_~MID_print.txt) 파일명 출력부분 추가
2. 출력행중에 순수하게 해당칼럼만 존재하는 자료와 주석부분제외하고 추출
-생략예시
1) a.col1 
2) col2
3) col3 as col_b
4) a.col1 = b.col22

-포함예시
함수가 들어간 경우나 칼럼이외의 내용이 포함된 경우
1) ,'' as col1
2) substr(col2, 1,2)
3) nvl(colt, '#')
4) case문
5) max문
6) 기타 함수나 가공이 들어간부분

--5차수정요청
1. 파일과 테이블생성시 mid를 끝에 붙여서 mid별로 생성하도록 수정
2. 추출행 결과가 없는 경우는 파일의 해당행생성및 테이블행 생성하지 않도록 수정
3. --chk 옵션에따라 추출제외된 행은 별도의 파일로 생성하는 내용 추가
4. [mid 검색디렉토리 MID하위디렉토리] 의 형식을 --mid aaa,bbb형식으로 처리하도록 수정
